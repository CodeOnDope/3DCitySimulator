3D City Simulator Documentation

Overview

The 3D City Simulator is an interactive web-based tool for creating and visualizing 3D urban environments with perspective controls. This application allows users to generate buildings with various shapes, materials, and properties, and to visualize them with different rendering modes and lighting conditions.

Getting Started

To begin using the 3D City Simulator:

  • Click the "Generate City" button to create a random city layout
  • Use the navigation tools to explore the 3D environment
  • Select individual buildings to modify their properties
  • Experiment with different view modes and visualization options

Interface Overview

The interface is divided into two main sections:

  • 3D Viewport: The main area displaying the 3D city model
  • Controls Panel: The right sidebar containing all control options

The Controls Panel includes several sections:

  • City Generation Controls
  • Building Properties
  • Perspective Controls
  • View Modes
  • Visualization Options
  • Lighting Controls
  • Navigation Tools

Building Creation and Editing

Creating Buildings

  • Click the "Generate City" button to create a random city layout
  • Click the "Randomize" button to generate a new random configuration

Building Shapes

The simulator supports various building shapes:

  • Cube: Standard cubic building
  • Rectangle: Rectangular prism with adjustable width, height, and depth
  • Cylinder: Cylindrical building with circular base
  • Cone: Conical structure with circular base
  • Pyramid: Pyramid structure with square base
  • Sphere: Spherical structure
  • Pentagon: Building with pentagonal base
  • Hexagon: Building with hexagonal base
  • Octagon: Building with octagonal base
  • Mixed: Combination of different shapes

Editing Building Properties

Select a building by clicking on it, then modify its properties:

  • Width: Adjust the building's width
  • Height: Adjust the building's height
  • Depth: Adjust the building's depth
  • Color: Change the building's color
  • Material: Select from different material types

Perspective Controls

The simulator provides tools to visualize and understand perspective:

Vanishing Point

  • Toggle the vanishing point visibility
  • Use the "Refresh Vanishing Point" button to align the vanishing point with the current camera view

Perspective Lines

  • Toggle perspective lines to visualize how objects align with the vanishing point
  • Adjust line color to improve visibility against different backgrounds

View Modes

The simulator offers several rendering modes:

  • Shaded: Standard solid view with lighting and shadows
  • Wireframe: Shows only the edges of the geometry
  • Wireframe on Shaded: Combines solid objects with visible wireframes
  • X-Ray: Transparent view that allows seeing through buildings
  • Silhouette: Shows buildings as silhouettes with outlines

To change the view mode, select the desired option in the View Mode section.

Material Properties

Buildings can be assigned different materials:

  • Standard: Default material with balanced properties
  • Pencil Sketch: Rough, flat-shaded material resembling hand drawing
  • Clay: Completely matte material with no reflections
  • Paper: Slightly transparent material with minimal shininess
  • Metal: Highly reflective material with low roughness
  • Glass: Transparent material with high shininess

To change a building's material, select the building and choose a material from the dropdown menu.

Lighting Options

The simulator provides extensive lighting controls:

Lighting Modes

  • Daylight: Bright, even lighting simulating daytime
  • Night: Dark environment with point lights
  • Sunset: Warm directional lighting with soft shadows

Lighting Properties

  • Light Intensity: Controls the brightness of the main directional light
  • Ambient Intensity: Adjusts the overall ambient lighting in the scene
  • Enable Shadows: Toggles shadows on/off
  • Shadow Intensity: Controls how dark the shadows appear
  • Diffuse Intensity: Adjusts the soft diffuse lighting

Navigation Tools

The simulator provides several ways to navigate the 3D environment:

Navigation Modes

  • Tumble: Rotate around the center point (Left mouse button + drag)
  • Pan: Move the camera parallel to the view plane (Middle mouse button + drag)
  • Dolly: Zoom in and out (Right mouse button + drag)

You can also use the navigation tool buttons in the interface to switch between modes.

Visualization Options

The simulator offers additional visualization controls:

  • Scene Brightness: Adjust the overall brightness of the scene
  • Scene Contrast: Modify the contrast level for better visual distinction
  • Bloom Effect: Add a subtle glow to bright areas for more realistic lighting
  • Fog: Enable atmospheric fog with adjustable density and color

Keyboard Shortcuts

  • R: Reset camera to default position
  • G: Generate new city
  • W: Toggle wireframe mode
  • S: Toggle shadows
  • F: Focus on selected building
  • Esc: Deselect current building

Troubleshooting

Performance Issues

If the simulator runs slowly:

  • Disable bloom effect and shadows for better performance
  • Reduce the number of buildings in the scene
  • Use simpler view modes like Shaded or Wireframe

Display Problems

If buildings or controls don't display correctly:

  • Ensure your browser is up to date
  • Try a different browser (Chrome, Firefox, or Edge recommended)
  • Check that JavaScript is enabled

Navigation Issues

If camera controls aren't working properly:

  • Ensure you're using the correct mouse button for each navigation mode
  • Try clicking the navigation tool buttons to reset the mode
  • Refresh the page if controls become unresponsive
